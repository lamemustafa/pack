# Code Of Conduct

ComplyEaze Pack follows the Contributor Covenant Code of Conduct, adapted for a
public project that handles sensitive compliance workflows.

## Our Standards

Examples of behaviour that contributes to a positive environment:

- using welcoming and inclusive language;
- respecting differing viewpoints and experience levels;
- giving and accepting constructive feedback;
- focusing on what is best for the community and users;
- showing empathy toward other contributors.

Examples of unacceptable behaviour:

- harassment, threats, insults, or personal attacks;
- sexualised language or imagery;
- publishing another person's private information without explicit permission;
- trolling or sustained disruption;
- posting real GSTIN, PAN, Aadhaar, taxpayer/client names, credentials, OTPs,
  CAPTCHA responses, cookies, downloaded files, portal screenshots containing
  tax data, portal HTML, or raw network responses in public project spaces.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues,
and other contributions that do not follow this Code of Conduct. Maintainers may
also temporarily or permanently ban contributors for behaviour that they deem
inappropriate, threatening, offensive, or harmful.

Report conduct issues to `contact@complyeaze.com`. For security vulnerabilities,
use [SECURITY.md](SECURITY.md) instead.
